- Nové verze VS Code přinášejí hlubší integraci GitHub Copilot (AI chat přímo v editoru, inline návrhy), což zrychluje generování AL boilerplate kódu a testovacích metod.
- Optimalizace výkonu LSP (Language Server Protocol), což má přímý dopad na rychlost kompilace a validace syntaxe na pozadí u velkých AL projektů (multi-root workspaces).
- Vylepšené možnosti ladění (debuggingu) a správy profilů spuštění v `launch.json` pro efektivnější připojování k BC cloud sandboxům a lokálním Docker kontejnerům.
