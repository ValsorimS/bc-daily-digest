Verdikt: NE – Článek obsahuje pouze odkaz na obecné testovací sestavení VS Code Insiders a neposkytuje žádné specifické informace pro vývoj v AL nebo Dynamics 365 Business Central.

<!--více-->

- Insiders verze VS Code přináší raný přístup k novým API pro GitHub Copilot a AI agenty, což umožňuje pokročilejší kontextovou analýzu AL kódu a generování AL testů.
- Optimalizace běhového prostředí extension hostu zlepšuje stabilitu a odezvu AL Language rozšíření při parsování rozsáhlých aplikačních objektů a načítání symbolů.
- Vylepšení v oblasti správy multi-root workspaces a debugování usnadňují paralelní vývoj a trasování kódu u komplexních, na sebe navazujích BC aplikací.
