Verdikt: ANO – Článek odkrývá budoucnost vývojových workflow ve VS Code, což přímo ovlivní efektivitu vývoje v AL pomocí autonomních GitHub Copilot agentů.

<!--více-->

- Integrace GitHub Copilot v autonomním "agent" módu, který dokáže samostatně analyzovat workspace, navrhovat refaktoring AL kódu a provádět komplexní změny napříč objekty.
- AI-native automatizace testování zaměřená na generování a validaci testovacích scénářů, aplikovatelná na AL Test Toolset a robustní CI/CD pipelines.
- Pokročilé AI-driven code review integrované přímo do vývojového cyklu, doplňující standardní AL lintery a AppSource Cop analýzu o sémantickou kontrolu kvality kódu.
