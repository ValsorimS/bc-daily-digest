Verdikt: ANO – Článek popisuje integraci MCP do VS Code, která zásadně posune možnosti AI asistentů při psaní a správě AL kódu.

<!--více-->

- VS Code zavádí podporu pro MCP (Model Context Protocol) Apps, což umožňuje AI agentům běžícím v IDE generovat a ovládat plně interaktivní uživatelské rozhraní (Webviews) namísto dosavadního omezeného textového výstupu v chatu.
- V kontextu AL vývoje tato technologie umožní vznik specializovaných BC agentů, kteří dokážou vizuálně prezentovat schémata tabulek, interaktivně řešit konflikty v kódu při upgradech nebo spravovat alokaci Object ID přímo přes UI integrované v chatu.
- Architektura MCP standardizuje API pro bezpečné obousměrné propojení LLM s lokálním vývojovým prostředím, což otevírá cestu pro těsnou integraci lokálního AL Language serveru, kompilátoru (alc.exe) a telemetrie Business Central s autonomními AI agenty.
