*   AL Object ID Ninja je nově dostupný jako GitHub Action a Azure DevOps Pipeline Task, umožňující přímou integraci do CI/CD pipeline pro AL repozitáře.
*   Nástroj automaticky skenuje AL repozitář během CI/CD procesu a identifikuje Object ID, Field ID a Enum Value ID, která nejsou evidována centrálním Ninja backendem.
*   Při detekci neschválených či nesledovaných ID systém cíleně selže CI/CD build, čímž striktně vynucuje dodržování ID rozsahu a eliminuje manuální správu ID.
