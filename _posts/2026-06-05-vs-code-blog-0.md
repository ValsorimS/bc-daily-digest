* Aktualizace VS Code API a povýšení interního Node.js runtime přímo ovlivňují stabilitu, výkon a kompatibilitu AL Language extension, zejména lokalizovaného Language Serveru (LSP).
* Vylepšení integrovaných AI schopností (GitHub Copilot / Copilot Chat) v edici Insiders přinášejí pokročilejší kontextové agenty, což usnadňuje analýzu rozsáhlých AL workspace a automatizaci generování testovacích codeunitů.
* Optimalizace jádra editoru, správy paměti a debuggeru ve VS Code zkracují odezvu při lokální kompilaci (Ctrl+Alt+F5) a usnadňují trasování kódu při debuggingu BC sandboxů.
