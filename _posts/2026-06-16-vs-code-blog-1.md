Verdikt: NE – Text obsahuje pouze obecné oznámení o vydání nové verze editoru VS Code bez jakýchkoliv specifických detailů pro vývojáře v AL či Dynamics 365 Business Central.

<!--více-->

- Každá aktualizace VS Code přímo ovlivňuje hostování AL Language Extension, zejména stabilitu Language Server Protocolu (LSP), rychlost indexace objektů a debugování Business Central sandboxů.
- Nové verze VS Code rozšiřují API pro AI agenty (např. GitHub Copilot), což vývojářům umožňuje efektivnější inline generování AL kódu, psaní unit testů a automatickou refaktorizaci AL objektů.
- Vylepšení správy paměti a multi-root workspaces v editoru optimalizují práci s rozsáhlými BC projekty obsahujícími desítky provázaných AL aplikací (app-to-app dependencies).
