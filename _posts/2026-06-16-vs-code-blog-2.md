Verdikt: NE – Analyzovaný text obsahuje pouze prázdný odkaz na novinky ve VS Code a nenabízí žádné konkrétní informace pro vývojáře v Business Central.

<!--více-->

- Aktualizace jádra VS Code přímo ovlivňují stabilitu a propustnost LSP (Language Server Protocol) pro AL Language Extension, což má dopad na rychlost parsování symbolů a odezvu IntelliSense u rozsáhlých BC projektů.
- Nové verze VS Code standardně rozšiřují rozhraní pro AI agenty (GitHub Copilot Chat / VS Code Speech), což umožňuje efektivnější kontextovou generaci AL kódu, psaní testovacích codeunitů a refaktorizaci legacy kódu.
- Změny v debugger protokolu (DAP) a správě multi-root workspaces optimalizují deployment a debugging v komplexních multi-app prostředích Business Central (správa závislostí mezi aplikacemi).
