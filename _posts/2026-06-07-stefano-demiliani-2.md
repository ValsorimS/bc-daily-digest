Verdikt: ANO – Článek je zásadní pro BC vývojáře navrhující pokročilé Copilot agenty, protože popisuje přechod z Chat Completions na efektivnější a flexibilnější Responses API v Azure OpenAI.

<!--více-->

- Nové Azure OpenAI Responses API sjednocuje generování textu, volání nástrojů (tool calling) a zpracování strukturovaných výstupů do jednoho unifikovaného rozhraní namísto dřívějších fragmentovaných API volání.
- V kontextu AL a Business Central Copilot toolkitu to zásadně zjednodušuje orchestraci AI agentů, nativně podporuje zachování stavu konverzace (statefulness) a eliminuje složitou správu historie zpráv v AL kódu.
- API přináší deklarativnější přístup k definici schémat pro strukturované odpovědi (Structured Outputs), což přímo usnadňuje parsování silně typovaných JSON dat v AL pomocí datových typů `JsonObject` a `JsonArray` při integraci s BC tabulkami a API stránkami.
