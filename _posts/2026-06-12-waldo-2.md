Verdikt: ANO – Článek představuje praktické nasazení horkého trendu AI agentů (MCP) pro usnadnění diagnostiky BC SaaS telemetrie bez nutnosti psaní KQL.

<!--více-->

- Nástroj „BC Telemetry Buddy“ implementuje otevřený standard MCP (Model Context Protocol) od Anthropic pro VS Code, což umožňuje LLM modelům přímý kontextuální přístup k datům telemetrie Business Central.
- Řešení funguje lokálně v rámci stávajícího chatovacího rozhraní (např. GitHub Copilot) bez nutnosti platit za externí API klíče, přičemž překládá dotazy v přirozeném jazyce na KQL (Kusto Query Language) pro Azure Application Insights.
- Tento koncept demonstruje posun k agentickému vývoji v ekosystému Dynamics 365, kde AI agent přebírá kognitivní zátěž spojenou se znalostí telemetrického schématu BC a syntaxe Kusto dotazů.
