<!--více-->

* Nové verze VS Code optimalizují Language Server Protocol (LSP), což má přímý dopad na výkon, rychlost indexace rozsáhlých AL projektů a stabilitu IntelliSense v AL Language Extension.
* Pokročilá integrace AI agentů (GitHub Copilot / VS Code Copilot Chat) přináší lepší kontextové porozumění AL syntaxi, což usnadňuje generování testovacích codeunit a refaktorizaci legacy kódu.
* Aktualizace API pro debugování a správu workspace profilů zjednodušují konfiguraci launch.json pro multi-root AL projekty a zefektivňují ladění kódu napříč více BC sandboxy.
