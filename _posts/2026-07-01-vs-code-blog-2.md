Verdikt: ANO – Článek je klíčový pro pochopení evoluce platformy VS Code, na které plně závisí AL vývojové prostředí, a přináší zásadní novinky v integraci AI copilotů.

<!--více-->

- **Nová éra AI agentů v editoru:** Rozšíření API pro GitHub Copilot Chat umožňuje vývojářům vytvářet specifické AL agenty, kteří dokážou analyzovat aplikační architekturu Business Central, generovat AL unit testy a refaktorovat legacy kód s hlubokým kontextem workspace.
- **Optimalizace Language Server Protocol (LSP):** Vylepšení výkonu na pozadí VS Code zrychluje odezvu AL Language Extension při indexování masivních projektů, načítání symbolů z Base Application a snižuje paměťovou náročnost při práci s monorepy.
- **Pokročilé nástroje pro debugging a Git:** Nová verze přináší nativní vizualizaci složitých merge konfliktů a vylepšené trasování volání (Call Stack visualization), což výrazně usnadňuje debugging AL kódu běžícího na lokálních BC Docker containerech a integraci upstream změn z Microsoft releasů.
