- OpenAI a VS Code v rámci dvoutýdenního experimentu optimalizovaly systémové prompty pro model GPT-5.5 v kódovacím prostředí za účelem zrychlení přímých úprav kódu.
- Úpravou promptů bylo dosaženo snížení počtu volání externích nástrojů (tool calls) a redukce spotřeby tokenů na konci generování (tail-end token usage), což minimalizuje latenci při interakci s AI.
- Pro AL vývojáře využívající AI agenty a Copilot ve VS Code to indikuje trend směřující k rychlejší a efektivnější generaci kódu na pozadí vývojového prostředí bez nutnosti navyšovat výkon lokálního hardwaru.
