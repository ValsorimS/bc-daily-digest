Verdikt: ANO – Novinky v AI agentech pro VS Code jsou vysoce relevantní pro AL vývojáře, neboť přímo ovlivní jejich pracovní postupy a efektivitu.

<!--více-->

*   Příchod "Agent Mode" pro VS Code signalizuje hlubokou platformní integraci AI agentů, což pro AL vývojáře implikuje potenciál pro agent-driven code generation, refactoring, linting a automatizovanou tvorbu či modifikaci AL objektů, page extension a table extension na základě kontextových dat a příkazů přímo v IDE.
*   Rozšířená integrace GitHub Copilot a uvedení GitHub Copilot CLI posiluje AI-assisted AL vývoj, umožňující skriptovatelnou automatizaci generování AL kódu, automatické doplňování komplexních AL konstrukcí (např. event subscribers, report datasety) a potenciální integraci do CI/CD pipeline pro automatické revize či optimalizace AL kódu.
*   Diverzifikace trhu s kódovacími agenty (OpenAI, Anthropic) otevírá cestu pro vznik vysoce specializovaných AL-aware agentů nebo možnost využít různé modely pro specifické AL výzvy, jako je optimalizace databázových operací v AL, generování komplexních datových modelů pro extensions či automatické přepisování starších AL kódů do moderních AL standardů.
