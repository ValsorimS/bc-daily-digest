- Aktualizace VS Code jádra přímo ovlivňují stabilitu a rychlost AL Language extension, zejména prostřednictvím optimalizací Language Server Protocol (LSP), což zrychluje IntelliSense a validaci kódu na pozadí u velkých projektů (AppSource aplikací).
- Nové verze editoru přinášejí hlubší integraci GitHub Copilot a nativních AI chat agentů do vývojového prostředí, což umožňuje efektivnější inline generování AL kódu, psaní ALDoc dokumentace a automatizované refaktorování.
- Vylepšení v oblasti správy paměti a renderingu velkých souborů ve VS Code optimalizují práci s objemnými soubory snapshotů (.alshot) při analýze výkonu a ladění BC aplikací přes Snapshot Debugging.
