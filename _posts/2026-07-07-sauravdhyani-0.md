- Business Central 2026 Release Wave 1 (BC28) vyžaduje pro on-premise nasazení operační systém Windows 11 nebo Windows Server 2022/2025 a minimálně 16 GB RAM.
- Aplikační platforma přechází na runtime .NET 8.
- Databázový subsystém vyžaduje MS SQL Server s nastavenou kompatibilitou (compatibility level) výhradně na hodnoty 160 nebo 170.
